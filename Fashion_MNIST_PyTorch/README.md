# Fashion-MNIST Image Classification (PyTorch)

This project uses **PyTorch** to train a **neural network** for image classification on the **Fashion-MNIST dataset**.

---

## Overview

- Framework: PyTorch  
- Dataset: Fashion-MNIST  
- Model: Fully Connected Neural Network (MLP)  
- Task: Image Classification  

---

## Model Architecture

- Input: 28 × 28 images (flattened)
- Hidden Layers: 512 → 512 (ReLU)
- Output: 10 classes
